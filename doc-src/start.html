Hello walrus: the simplest possible Cubictemp template

The most basic function of a templating system is text substitution - that is, taking one piece of text, and inserting it into another. Consider the following code snippet:

import cubictemp template = "Hello @_!foo!@." temp = cubictemp.Template(template) print temp(foo="walrus")

This is a complete Cubictemp example - it constructs a template, with a single substitution tag called "foo". It then calls the template, passing in a namespace within which the template is evaluated. The output of this program is: 

Hello walrus.

That's it - perhaps the simplest possible Cubictemp example.

Cubictemp throws the TempError exception whenever there is an error in an evaluated expression, or a syntax problem in a template. This exception exposes a number of attributes that are useful for debugging:

pos

The character offset of the problematic text from the beginning of the template.

lineNo

The line number of the problematic text.

temp

The offending template instance.

contextLen

A class variable that determines the number of lines of context to display (default 2). It can be changed as follows: cubictemp.TempError.contextLen = 5

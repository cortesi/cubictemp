Cubictemp is a small, elegant templating system for Python.

Features

  • Cubictemp provides enough power to make the common use cases as effortless as possible, but encourages good design by dis-allowing statements in templates.
  • Cubictemp integrates tightly with Python, allowing you to pass arbitrary Python objects into a template, walk sequences and iterators, and evaluate expressions.
  • Cubictemp includes default escaping, protecting you against common classes of HTML and Javascript Cross-Site Scripting vulnerabilities.
  • Cubictemp is tiny - only ~ 170 SLOC. This is a good thing. There are many large, over-designed Python templating systems out there. Cubictemp proves that a templating sytem can be complete, elegant, powerful, fast and small, all at the same time.
  • Cubictemp is simple, easy to use, well-tested and well-documented.

Requirements

Cubictemp requires Python 2.4 or better.
